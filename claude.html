GSAP Stagger Animations

Basic Stagger

Simple left-to-right stagger with 0.1s delay between elements
gsap.to(".box", { opacity: 1, duration: 0.5, stagger: 0.1 });

Stagger From Center

Animation radiates from the center
gsap.to(".box", { opacity: 1, duration: 0.5, stagger: { amount: 1, from: "center" } });

Grid Stagger

Radial stagger from center with easing
gsap.to(".grid-box", { opacity: 1, scale: 1, duration: 0.7, stagger: { amount: 1.5, from: "center", grid: "auto", ease: "power2.out" } });
